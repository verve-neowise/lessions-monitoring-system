# Groups

Route `org/:org_id/groups`

> Permissions: `groups`

> Header: `Authorization` : `<token>`

---

**`GET`** /

Get all groups

---

**`GET`** /:id

Get group details

---

**`POST`** /

Create group

```json
{
    "name": "Node.js 146",
    "months": 6,
    "directionId": 3 // Node.js
}
```

Response:

```json
{
    "message": "group created.",
    "group": {
        "id": 3,
        "name": "Node.js 148",
        "months": 6,
        "direction": {
            "id": 1,
            "name": "Backend Node.js",
            "status": "active"
        }
    }
}
```

---

**`PUT`** /:id

Update group

```json
{
    "name": "Node.js 146",
    "months": 6,
    "directionId": 3 // Node.js
}
```

Response:

```json
{
    "message": "Group updated.",
    "group": {
        "id": 3,
        "name": "Nodejs 148",
        "directionId": 1,
        "months": 6,
        "teacherId": null,
        "status": "active"
    }
}
```

---

**`DELETE`** /:id

Delete group

Response:

```json
{
    "message": "Group deleted.",
    "group": {
        "id": 3,
        "name": "Nodejs 148",
        "directionId": 1,
        "months": 6,
        "teacherId": null,
        "status": "deleted"
    }
}
```
